## Session 2 — Full Audit Fixes (2026-04-07)

### Mobile hero image quality
| # | File | What was fixed |
|---|------|---------------|
| 6 | `components/LiquidImage.tsx` | **Pixel ratio was hardcoded to `1` on mobile** — modern phones have 2×–3× displays so the WebGL canvas was rendered at half resolution and upscaled, causing blur. Changed to `Math.min(window.devicePixelRatio, 2)` for all devices |
| 7 | `components/LiquidImage.tsx` | Added `aria-label` + `role="img"` to the canvas container so screen readers can announce the image |

### Gallery page — was completely broken on mobile
| # | File | What was fixed |
|---|------|---------------|
| 8 | `app/gallery/page.tsx` | **Critical**: layout was `flex-row` with hard `w-1/2` panels and zero mobile alternative — page was unusable on phones. Restructured to `flex-col md:flex-row`: on mobile the main image takes `62vh`, controls stack below; close button moves onto the image overlay; next-image preview is hidden on mobile; number, PREV/NEXT, and counter all reposition correctly |

### Font sizes too small
| # | File | What was fixed |
|---|------|---------------|
| 9 | `app/page.tsx` | Footer: `text-[8px]` → `text-[10px]` (WCAG minimum ~12px; increased toward that) |
| 10 | `app/projects/page.tsx` | All `text-[8px] md:text-[10px]` labels (Scroll, Commercial Interiors, ©2026) → `text-[10px] md:text-[11px]` |
| 11 | `app/projects/page.tsx` | "Drag to explore" hint: `text-[8px]` → `text-[10px]`, opacity bumped from `0.30` → `0.40` |
| 12 | `app/contact/page.tsx` | Footer: `text-[8px] md:text-[10px]` → `text-[10px] md:text-[11px]` |
| 13 | `app/gallery/page.tsx` | All control labels bumped to minimum `text-[10px]` |

### Contact form — was completely non-functional
| # | File | What was fixed |
|---|------|---------------|
| 14 | `app/contact/page.tsx` | `onSubmit` previously just called `e.preventDefault()` — nothing was sent. Now collects Name / Email / Message into state, constructs a `mailto:` URL, opens the user's email client pre-filled, and shows an animated "Thank you" confirmation screen with a "Send another message" reset button |
| 15 | `app/contact/page.tsx` | Added `<label htmlFor>` elements (screen-reader accessible, visually hidden via `sr-only`) for all three form fields |
| 16 | `app/contact/page.tsx` | Added `required` attributes + `focus-visible` outline on submit button |

### Accessibility & meta
| # | File | What was fixed |
|---|------|---------------|
| 17 | `app/layout.tsx` | Added `export const viewport` with `width: device-width`, `initialScale: 1`, `maximumScale: 5`, `themeColor: #141414` — was missing entirely |
